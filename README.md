# Wordle RN

React Native (Expo) Wordle-style game with navigation, multimedia, Android-specific features, and tests.

## Scripts
- `npm start` — start Expo dev server
- `npm run android` — run on Android
- `npm run ios` — run on iOS
- `npm test` — run unit tests (ts-jest)

## Features
- React Navigation (Home, Game, Media)
- Wordle gameplay (6x5 grid, on-screen keyboard, evaluation)
- Multimedia: images (expo-image), audio/video (expo-av)
- Android: ToastAndroid, BackHandler, Vibration
- Tests: ts-jest unit tests for logic
